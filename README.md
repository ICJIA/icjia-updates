# ICJIA News & Updates

## Install

```
$ git clone https://github.com/ICJIA/icjia-updates.git
$ cd icjia-updates
$ npm install
```

## Develop

```
npm run serve
```

## Build

```
npm run build
```

## Deploy

Publish directory: `./dist`
